# 2. 탐색적 데이터 분석 (EDA)

## 🔍 데이터 탐색
- 기본 통계량 확인 (mean, median, std 등)
- 결측치, 이상치 유무 확인

## 🧹 데이터 정제 및 전처리
- 결측치 처리 (제거, 대체 등)
- 이상치 제거
- 형 변환, 인코딩, 정규화 등

## 📊 데이터 시각화
- 히스토그램, 박스플롯, 산점도 등 활용
- 카테고리별 분포 및 추세 파악

## 📈 일변량, 이변량, 다변량 분석
- **일변량 분석**: 변수 하나에 대한 분포 분석
- **이변량 분석**: 두 변수 간의 상관관계 파악
- **다변량 분석**: 세 개 이상의 변수 간 관계 분석

# 3. 데이터 분석

## 🧠 분석 기법
- 회귀분석 (선형/로지스틱)
- 클러스터링 (K-Means, DBSCAN 등)
- 분류 모델 (Random Forest, SVM, Neural Network 등)
- 시계열 분석 또는 PCA 등 (필요 시 추가)
